This collection of programs and files is used to update the Microsoft Access and SQL Server databases that receive information from Drilling Info.

The purpose of this is to reduce the amount of time needed to enter data to be used for internal records.

The Database Control file will run any process needed to keep everything up to date. Simply double click the program to begin. 

--------------------------------

For a more in depth explanation, check the "Database Package Explination.txt" file in the Documents folder.
